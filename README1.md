# Desc. del proyecto (resumen)
Plataforma web centralizada para la administración de inventario y soporte técnico del Laboratorio LS5 del Instituto Tecnológico de Saltillo. El sistema automatiza el control de préstamos, devoluciones y la auditoría física de equipos mediante escaneo de códigos QR.

# Tecnologías utilizadas (Arq Software)
* **BACKEND:** Java
* **FRONTEND:** HTML5, CSS3, JavaScript (Bootstrap)
* **BD:** MySQL
* **Serv.:** Ubuntu Server (Linux)
* **API:** Librería Html5Qrcode (para activar la cámara del dispositivo)
